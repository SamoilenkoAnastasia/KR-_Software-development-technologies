##  Огляд 
**«Особиста бухгалтерія»** — це десктопний інструмент для управління персональними та спільними фінансами.

## Інструкція із запуску

Для успішного запуску проєкту необхідно виконати підготовчі кроки для налаштування середовища.

### 1. Передумови

1.  **Java Development Kit (JDK):** Встановлена версія 17 або вище.
2.  **Maven:** Встановлена система управління збіркою.

### Підготовка Проєкту

Завантажте проєкт як ZIP-архів і розпакуйте його:
Використовуйте термінал, перебуваючи у кореневій директорії проєкту, для збірки та запуску через Maven:
Збірка проєкту (завантаження залежностей): mvn clean install
Запуск застосунку (JavaFX): mvn javafx:run
